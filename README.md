# Tomato Plant Disease Detection using Deep Learning

## Project Description
This project detects tomato plant diseases from leaf images using a deep learning CNN model (ResNet18). 
After prediction, the system provides crop advisory including causes, prevention methods, and treatment suggestions.

## Technologies Used
- Python
- PyTorch
- CNN (ResNet18)
- Transfer Learning
- Gradio
- Computer Vision

## Dataset
Dataset used:
https://www.kaggle.com/datasets/anamikachhabra/leaf-based-tomato-plant-disease-recognition-data

Total Images: ~16,000
Classes: 10

## Steps to Run

1 Install dependencies

pip install -r requirements.txt

2 Train model

python train_model.py

3 Run web application

python app.py

## Features
- Automatic tomato disease detection
- Image classification using CNN
- AI crop advisory
- Web interface using Gradio

## Example Diseases
- Tomato Early Blight
- Tomato Late Blight
- Tomato Mosaic Virus
- Tomato Yellow Leaf Curl
- Healthy Tomato Leaves
